About Anime Dashboard

Sometimes choosing an anime to watch can be hard work.

Anime Dashboard makes it easier by showing you all the information you need, in order to choose something you want to watch!

The information focuses on a few important factors. Popularity, average user rating, genre, type (movie, tv etc...) and number of episodes.

The Dashboard Explained

1. The line chart compares how popular a show is, by number of members in it's fan group with the average user rating. Click and drag the chart to select anime with the ratings you want to see.

2. Click on a slice in the pie chart to select which type of anime you would like to look for. If the slice is filled with colour, you will be able to see it's respective type in the select menu.

3. Interacting with the line and pie charts will filter the results in the select menu. Scroll through the select menu and click the one you would like to know more about. **To undo selections from any of the other charts, scroll to the top of this menu and click 'select all**'.

4. After you choose an anime from the select menu, you will be able to see it's information in the table.
